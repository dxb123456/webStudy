  1. 一个参数：0 或者1 ,代表 grow和shrink为0 或者1 ，basis：0%
  2. 一个参数：百分比或者长度，代表basis为百分比，growhe shrink为1
  3. 两个参数：两个数字，一个grow 一个shrink，basis为0%
  4. 两个参数有：一个数字，一个百分比或者长度，数字表示grow 、百分比或者长度表示basis,shrink 默认值是缩放

flex 是flex-grow、flex-shrink、flex-basis的缩写；

flex:none == 0 0 auto

flex:auto == 1 1 auto

1个非负数字， flex:0 == 0 0 0%

1个非负数字，flex:1 == 1 1 0% 效果与auto类似

1个长度或者百分比 flex：0% === 1 1 0%

两个非负数数字 flex:2 3 === 2 3 0%

一个非负数字 和 一个长度或者百分比 flex:10 333px == 10 1 333px
