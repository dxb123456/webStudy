福奇說美國可以採取行動提供第三針疫苗2021.08.16
全文共830字，讀完約需3分鐘

福奇表示如果有必要，準備迅速向更廣泛的人群提供第三劑疫苗

表示衛生官員正在「每天和每週」評估各類人群。

美國總統拜登的首席醫學顧問福奇表示，如果有必要，美國「絕對準備」迅速向更廣泛的人群提供第三針冠狀病毒疫苗。

他沒有給出時間表，表示衛生官員正在「每天和每週」評估各類人群。

「因此，如果接下來的資料證明我們確實需要向療養院的人或者老年人提供額外的劑量，我們絕對準備非常迅速地行動，」他在哥倫比亞廣播公司的「Face the Nation」節目中說。
